#README

This folder contains the following python files:

- merge_sort.py
- insertion_sort.py
- bubble_sort.py
- test_sorting_algoritms.py
- timing_sorting_algorithms.py
- binary_search.py


merge_sort.py contains a python implementation of the merge sort sorting algorithm

bubble_sort.py contains a python implementation of the bubble sort sorting algorithm

insertion_sort.py contains a python implementation of the insertion sort sorting algorithm

test_sorting_algorithms.py contains unit tests to test bubble_sort(), merge_sort(), and insertion_sort() to ensure the functions are correct

timing_sorting_algorithms.py times merge_sort(), bubble_sort(), and insertion_sort() and displays the time it takes each algorithm to sort a list of 1000 intengers

binary_search.py contains a python implementation of the binary search algorithm
