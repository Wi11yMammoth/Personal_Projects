
# README

In this lab I had to implement the following functions

See singly_linked_list.c

int count() - returns the number of nodes in the linked list
int max() - returns the largest integer in the linked list
int index() - returns the index position of the first integer in the linked list equal to the target
void extend() - adds the linked list values from one linked list to another
intnode_t * pop() - returns and deletes the first node in the linked list
